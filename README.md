# Mapbox-unity-sdk
### For Unity 2020.3+

To install the Mapbox Unity SDK package, follow these steps:
- In your Unity project, go to Windows > Package Manager.
- On the status bar, select the Add (+) button.
- From the Add menu, select "Add package from git URL". URL field will appear.
- In the URL field, enter the clone url of this repository.
- Select Add.
- The Editor will install the latest available version of the package and any dependent packages.
